Aide pour grand-père — site statique
Fichiers:
- index.html
- style.css
- script.js

Utilisation locale:
- Ouvre index.html dans un navigateur (double‑clic).

Hébergement (GitHub Pages):
1. Crée un nouveau dépôt GitHub.
2. Ajoute ces fichiers à la racine.
3. Dans les paramètres du dépôt -> Pages -> Source : main branch / root.
4. Le site sera disponible à <username>.github.io/<repo> après quelques minutes.

Jeu de données:
Le site contient une collection de raccourcis et astuces pour Windows et Chrome. Modifie 'script.js' pour changer le contenu.
